SABnzbd Client Support

Getting Started

To use this app, you need a running SABnzbd server with API access enabled.

  1. Open your SABnzbd web interface
  2. Go to Config → General
  3. Copy your API Key
  4. In the app, tap Settings → Add Server
  5. Enter your server details and API key

Common Issues

Can't connect to server: Ensure your device is on the same network as your SABnzbd server, or that your server is accessible remotely.

Invalid API key: Double-check the API key in SABnzbd's Config → General settings.
